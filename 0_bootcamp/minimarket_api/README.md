## Description

First Bootcamp project, Dockerizing a NondeJS (using NestJS) API with MongoDB.

## Installation

```bash
# Clone the repository
$ git clone
# Run Docker-compose
$ docker-compose up -d
# Install dependencies
$ yarn install --frozen-lockfile
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
